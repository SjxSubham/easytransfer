### Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (icons)

### Run Locally

```bash
git clone https://github.com/yourusername/easytransfer.git
cd easytransfer
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and you're good to go.
